🚀 轻量级 AI 聊天插件

PopupChatKit

一行代码,让你的网站拥有智能对话能力

📦 快速开始

只需要在你的网页中添加以下代码:

<!-- 引入 PopupChatKit 插件 -->
<script src="popup.js"></script>

<!-- 初始化插件 -->
<script>
  PopupChatKit.init({
    apiBase: 'http://localhost:8000/api'
  });
</script>

就是这么简单!现在点击右下角的悬浮按钮试试看吧 👉

⚙️ 配置选项

PopupChatKit.init({
  // API 基础地址
  apiBase: 'http://localhost:8000/api',

  // 按钮位置: 'bottom-right' | 'bottom-left' | 'top-right' | 'top-left'
  position: 'bottom-right',

  // 按钮大小(像素)
  buttonSize: 60,

  // 按钮颜色
  buttonColor: '#6366f1',

  // 聊天窗口最大宽度(像素)
  maxWidth: 400,

  // 聊天窗口最大高度(像素)
  maxHeight: 600,

  // 层级
  zIndex: 9999
});

🔧 API 方法

控制方法

PopupChatKit 提供了以下方法供你调用:

// 打开聊天窗口
PopupChatKit.open();

// 关闭聊天窗口
PopupChatKit.close();

// 切换显示/隐藏
PopupChatKit.toggle();

🛠️ 高级用法

事件监听

你可以通过自定义按钮触发聊天窗口:

<button onclick="PopupChatKit.open()">咨询客服</button>
