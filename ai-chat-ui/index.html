📖 设计说明

此页面展示了 PopupChatKit 所有功能页面的 UI 原型。采用网格布局,每个卡片代表一个独立页面。 设计遵循现代 Web 设计规范,注重用户体验和视觉美观。所有页面均为静态原型,仅用于开发参考,不含实际交互功能。
