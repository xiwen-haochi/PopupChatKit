## 📋 发布步骤

### 1. 初始化 Git（如果还没有）
```bash
cd PopupChatKit
git init
git add .
git commit -m "Initial commit: PopupChatKit v1.0.0"
```

### 2. 创建 GitHub 仓库
1. 访问 https://github.com/new
2. 仓库名: PopupChatKit
3. 描述: 一个轻量级、零依赖的 AI 聊天插件
4. 选择 Public
5. 不要初始化 README（我们已经有了）

### 3. 推送到 GitHub
```bash
git remote add origin https://github.com/yourusername/PopupChatKit.git
git branch -M main
git push -u origin main
```

### 4. 配置仓库
- 添加 Topics: ai, chatbot, javascript, fastapi, python, vue

## 🚀 快速开始

\`\`\`bash
# 克隆项目
git clone https://github.com/yourusername/PopupChatKit.git
cd PopupChatKit

# 配置环境
cp backend/.env.example backend/.env
# 编辑 .env 添加你的 ZHIPU_API_KEY

# 安装依赖
cd backend
uv sync

# 启动服务
uv run uvicorn main:app --reload --port 8000
\`\`\`

访问 http://localhost:8000 开始使用！

## 🔧 需要清理的内容

运行以下命令清理项目：

```bash
# 删除数据库文件
rm -f data/*.db

# 删除 .env 文件（保留 .env.example）
rm -f backend/.env

# 删除 ai-chat-ui 文件夹（如果不需要）
rm -rf ai-chat-ui

# 清理 Python 缓存
find . -type d -name "__pycache__" -exec rm -rf {} +
find . -type f -name "*.pyc" -delete
```
